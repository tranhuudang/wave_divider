## Usage

The `WaveDivider` widget provides a simple wave divider that you can customize. Here are some usage examples:

1. Using the default divider:
```dart
   WaveDivider()
```

or a little bit of custom with:
```dart 
    WaveDivider(
      thickness: 3,
      color: Colors.amber,
      waveHeight: 7,
      waveWidth: 14,
    )
```
